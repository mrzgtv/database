1. В Dbeaver создаем новую вкладку.
Прописываем следующий код:

[client]
user=root
password=*заданный пароль*

Сохраняем файл под именем my.cnf на диске С:\.

Вариант для Windows.

При запуске в командной строке утилиты mysql пароль больше не нужно вводить.

2. В Dbeaver нажимаем "Новый редактор SQL".
В открывшемся окне вводим следующий код:
Create database example;
Use example;
Create table users(
	id int,
	name varchar(100)
);

Выделяем код, нажимаем ALT+X. База данных и таблица созданы.

3. Для дампа используем командную строку и утилиту mysqldump.

Mysqldump example > sample.sql
	     ^		^
Example - назывние базы данных которое хотим скопировать.
Sample.sql - файл, куда выводим скопированную базу.

Для разворачивания дамп-файл в командной строке используем команду:

Mysql sample < sample.sql
Sample - куда разворачиваем
Sample.sql - исходная копия, откуда берем данные.
